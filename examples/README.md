# Examples

This directory contains example scripts demonstrating how to use NoxRunner.

## Quick Start Example

See `quickstart.py` for a complete working example that demonstrates:

- Creating a sandbox
- Executing commands
- Uploading and downloading files
- Managing sandbox lifecycle

## Running Examples

```bash
# Make sure you have a NoxRunner backend running
# (e.g., the sandbox manager from the sandbox project)

# Run the quickstart example
python examples/quickstart.py
```

## Environment Variables

- `NOXRUNNER_BASE_URL`: Base URL of the NoxRunner backend (default: `http://127.0.0.1:8080`)
